## 🌟 ¿Qué es OpenCMO?

OpenCMO es un **ecosistema de marketing de IA multiagente** adaptado para equipos pequeños o product makers. Simplemente proporciona la URL de tu producto y OpenCMO:
1. **Analizará tu sitio web** para comprender a tu público.
2. **Organizará un debate de estrategia** entre varios agentes para identificar las mejores palabras clave y posicionamiento.
3. **Automatizará el monitoreo continuo** en SEO, visibilidad (GEO) y comunidades de desarrolladores.

## 🕸️ Grafo de Conocimiento Interactivo

El **Grafo de Conocimiento** convierte datos abstractos en un impresionante diagrama de red dinámico visual.

<div align="center">
  <img src="assets/knowledge_graph_demo.png" alt="Marketing Knowledge Graph" width="850" style="border-radius: 8px; box-shadow: 0 4px 8px rgba(0,0,0,0.5);" />

### Atributos Principales:
- 🔵 **Exploración Interactiva**: Zoom e interacción a través del universo digital de tu marca.
- 🟢 **6 Dimensiones de Nodos**: Diferencia visualmente entre tu Marca, Palabras clave, Discusiones, Clasificaciones SERP y Competidores.
- ⚡ **Sincronización en Tiempo Real**: Se equilibra automáticamente cada 30 segundos conforme ingresan nuevos datos.

## ⚙️ Guía de Inicio Rápido

```bash
git clone https://github.com/study8677/OpenCMO.git
cd OpenCMO

# Instalar dependencias
pip install -e ".[all]"

# Configurar variables
cp .env.example .env

# Lanzar Dashboard Web
opencmo-web
```
🚀 Entra a [http://localhost:8080/app](http://localhost:8080/app) en tu navegador.
